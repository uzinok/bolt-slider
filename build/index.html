BOLT-SLIDER набор плагинов для создания слайдеров на сайте.

  • Описание проекта

  • Простой слайдер. Слайдер с минимальным количеством опций.

Описание.

Стили только необходимые для работы слайдера. В библиотеке нет стилей для кнопок навигации и пагинации. Элементы управления могут располагаться в любом месте кода. Кнопки навигации передаются элементами. Для создания кнопок пагинации передается блок в котором они будут генерироваться.

Если не переданы или с ошибкой переданы опции в консоль выводинтся ошибка:

⚠ Not slider options
⚠ Not slider

Доступность.

  1. для слайдера необходимо добавить атрибут 'aria-label'.
    Этот атрибут не должен содержать слово слайдер или карусель. так как скрин-ридер озвучит 'aria-roledescription', значение которого можно изменить опционально. По умолчанию 'carousel'.

  2. слайдам так же необходимо добавить атрибут 'aria-label'.
    Так же в 'aria-label' не нужно прописывать слово слайд так как скрин-ридер озвучит 'aria-roledescription', значение которого можно изменить опционально. По умолчанию 'slide'.
